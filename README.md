# strongest-blockchain-python
# 🛡️ Secure Land Registry Blockchain
This project is a decentralized solution for managing land records securely using Python and Blockchain technology.

## 🚀 Key Features
- **Tamper-proof:** Once data is entered, it cannot be changed.
- **Transparency:** All transactions are verifiable.
- **Zero Cost:** Built using open-source tools.

## 🛠️ Built With
- Python
- JSON for Data Storage
- Streamlit (for Frontend)
